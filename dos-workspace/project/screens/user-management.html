About roles

A quick reference for what each role can do across the workspace.

Owner

  • Full workspace control
  • Manage billing & seats
  • Create & archive projects
  • Manage roles & vendors
  • Edit all screeners & quotas

Editor

  • Create projects
  • Edit projects they're on
  • Write screeners & quotas
  • Manage billing
  • Change workspace settings

Viewer

  • Read project summaries
  • See field progress
  • Comment on screeners
  • Edit anything
  • Invite members
