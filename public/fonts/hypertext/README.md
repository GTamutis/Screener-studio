# Hypertext brand fonts

Drop licensed Hypertext `.woff2` files here (exact names required):

- `Hypertext-Regular.woff2` — body / UI (400)
- `Hypertext-Medium.woff2` — labels / UI emphasis (500)
- `Hypertext-Bold.woff2` — strong emphasis (700)
- `Hypertext-DisplayMedium.woff2` — headlines (500)

Until these files are present, the app falls back to Arial per Day One brand guidance.
